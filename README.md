# Online Tourist Guide System

A Java-based smart tourism application that provides tourist information, basic recommendations, and location-based support.

## Tech Stack
- Java
- JDBC
- MySQL
- HTML, CSS, JavaScript

## Features
- User Registration & Login
- Tourist Place Listing
- Category-based Recommendations
- JDBC Database Connectivity

## How to Run
1. Create MySQL database and tables
2. Update DB credentials in DBConnection.java
3. Run Java classes
